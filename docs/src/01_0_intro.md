# Introduction

This book is designed to guide you through practical, hands‑on examples of using **Rust** on **embedded systems**, with a particular focus on **Espressif microcontrollers**.

Its goal is to provide clear and well‑structured demonstrations of how to build reliable and efficient firmware using Rust.

Throughout this book, you will find

- Guidance on toolchain installation and project setup
- Examples covering peripherals, connectivity, and common embedded patterns
