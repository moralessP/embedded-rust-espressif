# Hardware Specifications

ESP32-S3 is a low-power MCU-based system on a chip (SoC) with integrated 2.4 GHz Wi-Fi and Bluetooth®
Low Energy (Bluetooth LE). It consists of high-performance dual-core microprocessor (Xtensa® 32-bit LX7), a
ULP coprocessor, a Wi-Fi baseband, a Bluetooth LE baseband, RF module, and numerous peripherals.

## ESP32-S3 board

![esp32-s3](images/ESP32-S3-DevKit.png)

- CPU and Memory
    - Xtensa® dual-core 32-bit LX7 microprocessor
    - Clock speed up to 240 Mhz
    - SRAM 512 KB
    - SRAM in RTC: 16 KB
- Connectivity
    - Wi-Fi (2-4 GHz band)
    - Bluetooth 5 LE
- Multiple peripherals
    - UART, I2C, I2S, SPI
    - Timers
    - 12-bit SAR ADC
    - Programmables GPIOs
- Security
    - Secure boot
    - Flash encrytpion

For more details, refer to the offcial documentation on the [Espressif](https://docs.espressif.com/projects/esp-hardware-design-guidelines/en/latest/esp32s3/product-overview.html) website.
